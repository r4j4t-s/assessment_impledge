The src folder has the module "compound_words". 
It contains the files "Trie.java", "Node.java" and Runner.java
The java files can be downloaded and compiled and ran via terminal or through an IDE like Eclpse.
The Runner.java is the class with main method where the actual solution exists.

//Configureation
